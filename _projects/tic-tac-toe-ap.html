TicTacToe (Advanced Programming) ❌/⭕️

TicTacToe is (*visually ugly*) a videogame created for the Advanced Programming course at UniPi (for the 21/22 academic year). The game is a simple TicTacToe implemented using Java Beans and Java AWT.
