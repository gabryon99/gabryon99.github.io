    Abstract: MeshLab, and its Python counterpart PyMeshLab, have been widely used for processing 3D scanned data. In this thesis, we have improved the capability of MeshLab to handle different kinds of 3D scanned data by adding the support for 3D scanned LIDAR data (E57).
